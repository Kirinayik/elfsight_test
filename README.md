## Installation and Setup Instructions

#### Example:  

Clone down this repository. You will need `node` and `npm` installed globally on your machine.  

Installation:

`git clone https://github.com/Kirinayik/elfsight_test.git`

`npm install`  

To Run Test Suite:  

`npm run test`  

To Start Server:

`npm run start`  

To Visit App:

`localhost:3000`
